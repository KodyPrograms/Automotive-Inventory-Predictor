Application Documentation

How to Install Python on Windows

  1. Go to the official Python website: https://www.python.org/downloads/windows/.
  2. Download the latest Python installer for Windows.
  3. Run the installer and check the box that says "Add Python to PATH".
  4. Click on Install Now and follow the setup wizard.
  5. After installation, verify it by opening a command prompt and typing:
    6. python --version

How to Generate the `.pkl` Model

The `.pkl` model stores the trained machine learning model, which is essential for making future predictions. Follow these steps to generate it:

  1. Ensure that the `cleaned_car_data.csv` file exists in the `csv/` folder.
  2. Run the following command:
    3. python mml.py
  3. If successful, the model will be saved as `vehicle_stock_forecast_model.pkl` in your project folder.

How to Clean Data

Use the cleaning script to process raw data and generate a cleaned version.

python clean_car_data.py

The cleaned data will be saved in the `csv/` folder as `cleaned_car_data.csv`.

How to Start the Program

  1. Ensure all dependencies are installed by running:
    2. python install.py
  2. Launch the application with:
    3. python app.py
  3. Open your browser and go to http://127.0.0.1:5000.

Troubleshooting

  • Issue: "No file selected" when uploading data.
    Solution: Ensure you have selected a `.csv` file.
  • Issue: "KeyError: 'Date'" in `mml.py`.
    Solution: Verify the data contains `Year` and `Month` columns.
  • Issue: "Module not found" error.
    Solution: Install dependencies using:
    python install.py

Features

  • Forecast Sales: Predicts how many units of each car model will be sold next month.
  • Buyer Insights: Analyzes buyer demographics, including gender and income.
  • CSV Download: Allows users to download the forecast report.
